Browser

   - It is a software tool used by clients to view webpages. 

   - Popular browsers

           a) Safari

           b) Edge

           c) Chrome

           d) FireFox 

           e) Opera etc.



 Browser Architecture

  1. UI [User Interface]

     - It refers to browser interface with elements like buttons, textbox, extensions etc.

     - Every browser can have different UI elements. 



  2. UI Backend 

      - It comprises of the logic required to control the UI. 

      - You can customize UI logic if it is an open source browser. 

   

  3. Browser Engine 

       - It is responsible for translating HTML & CSS in browser. 

       - Every browser can use a different engine. 

       - The popular browsers & their engines



                   Chrome        : V8

                   Edge            : Chakra

                   FireFox        : Spider Monkey, Gecko 

                   Safari            : Webkit 

                   etc.



  4. Rendering Engine 

       - It is responsible for rendering output into browser.        - Browser engine translates & rendering engine presents. 



  5. Javascript Interpreter

       - It is responsible for translating Javascript in browser. 

       - Interpreter translates line-by-line. 

       

  6. Network 

       - It is responsible for locating the resources required for web page.

       - It loads the resources into browser memory. 



  7. Data Persistance 

       - It is browser storage.

       - It store client details 

           a) Local Storage

           b) Session Storage

           c) Cookies etc.



   Note: Every browser follows “Netscape” architecture.
