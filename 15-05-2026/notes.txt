                                                 Main Semantic 

       - Main is considered as “Entry Point” in a computer program. 

       - Web page can also have an entry point for new users. 

       - Page can have multiple entry points. 

       - It is designed by using <main> container. 



       Syntax:    

               <main>

                   … entry point for new users … 

               </main>



       CSS Colors

       1. Color Name

               

                   {

                     color: red;

                   }



       2. Shade Name



                   {

                    color: lightyellow;

                   }



       3. Hexadecimal 

   

                   {

                    color: #f0f  or  #ff00ff;

                   }



       4. rgb()

               - It is a color with Red, Green and Blue combination. 

               - Every color can range from 0 to 255. 



               Syntax:

                   {

                    color: rgb(redValue, greenValue, blueValue);

                    color: rgb(255,0,0);

                   }



       5. rgba()

               - It red, green, blue and alpha.

               - “Alpha” sets transparent color range from 0 to 1. 



               Syntax:

               {

                color: rgba(255,0,0,0.5);

               }

   

       6. hsl()

               - It refers to hue, saturation and lightness. 

               - Hue is an angle from 0 to 360 deg.                        0 to 120        Red

                       121 to 240    Green

                       241 to 360    B lue 

               - Saturation is gray % in color from 1 to 100% 

                       1% is full gray.

                       100% is removing gray from color 

               - Lightness is brightness from 1 to 100%

               

           Syntax:

               {

                color: hsl(hue, sat, light);

                color: hsl(10deg, 50%, 60%);

               }



        7. hsla()

            - It is hue, saturation, lightness and alpha 

            - alpha is from 0 to 1.



         Syntax:

           { 

             color: hsla(10deg, 50%, 50%, 0.6);

           }



       8. Gradient Colors

           - You can apply gradient only by using “background-image”. 

           - Gradient is a combination of multiple colors in specific orientation.



               a) linear-gradient 

               b) radial-gradient 

       

           Syntax:

               {

                background-image: linear-gradient() or radial-gradient();

               }



            Syntax:

                linear-gradient(direction or angle, color1, color2..);

                linear-gradient(30deg, red 50%, geen, yellow);

                linear-gradient(to top right, red, yellow);
